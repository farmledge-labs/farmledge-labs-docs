# farmledge-docs

Public documentation for the Farmledge protocol — warehouse receipt tokenisation, commodity grading standards, custodian onboarding, lender API reference, and farmer guides.

## Run locally

```bash
pip install -r requirements.txt
mkdocs serve
```

Open http://127.0.0.1:8000 in your browser.

## Contributing

1. Fork this repository.
2. Create a branch: `git checkout -b docs/your-topic`.
3. Edit or add Markdown files under `docs/`.
4. Open a pull request against `main`.

Please keep content factual and concise. All pages must have at least a top-level heading.
